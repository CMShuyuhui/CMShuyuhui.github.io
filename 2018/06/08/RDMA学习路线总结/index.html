简介–什么是rdma

RDMA%28Remote%20Direct%20Memory%20Access%29%u6280%u672F%u5168%u79F0%u8FDC%u7A0B%u76F4%u63A5%u6570%u636E%u5B58%u53D6%u3002%0A%0ARDMA%20%28Remote%20DMA%29%20is%20the%20ability%20of%20accessing%20%28i.e.%20reading%20from%20or%20writing%20to%29%20memory%20on%20a%20remote%20machine%20withoutinterrupting%20the%20processing%20of%20the%20CPU%28s%29%20on%20that%20system.

RDMA的好处与优势：

1、零拷贝(zero-copy) RDMA transfers data directly from user virtual memory on one node to user virtual memory on another node, tcp copies into/out of system buffers on both nodes.

4、Message boundaties preserved. rdma是基于消息模式的，本身保留了消息边界，不同于tcp/ip流传输方式，需要做拆包解包操作。

rdma对于高性能的网络通讯来说，优势很多，但学习成本以及相关推广成本也是蛮高的，后面分享下本人摸索过程中的一些资料。
